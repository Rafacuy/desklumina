# Roadmap

This file describes features and ideas that are **not yet implemented**. Do not treat any item below as available functionality.

## Short-Term (v1.x)

- [ ] **Enhanced Visual Tool Display**: Improve the visual presentation of tool outputs in the UI to be more scannable and user-friendly.
- [ ] **Local LLM Integration**: Support for local AI inference engines like Ollama or Llama.cpp for privacy-conscious users who want to run the full stack locally.
- [ ] **Offline TTS**: Expand the text-to-speech engine to support offline models in addition to the current Edge TTS pipeline.

## Mid-Term (v2.x)

- [ ] **Custom Plugin System**: Allow users to load custom JavaScript or TypeScript tools dynamically without modifying the core project files.
- [ ] **Workflow Recording**: Record a series of desktop actions and replay them using a natural language alias.
- [ ] **Screen Awareness**: Allow the AI to "see" your screen via computer vision and describe its contents or interact with visual UI elements.

## Long-Term (Visionary)

- [ ] **Proactive Assistance**: DeskLumina should intelligently suggest actions based on your current workflow or calendar events without waiting for a prompt.
- [ ] **Native Mobile App**: Send commands to your desktop remotely via a secure mobile interface.
- [ ] **Full Home Automation**: Deep integration with Home Assistant to control your physical environment directly from your Linux desktop environment.
